Módulos

Todo lo que necesitas,
en un solo sitio

11 módulos integrados para cubrir las tareas más habituales de un técnico de sistemas Windows.

👤
Gestión de usuarios
Perfiles de C:\Users, usuarios locales, crear, eliminar, activar/desactivar y cambiar contraseña.
⚙️
Servicios Windows
Lista completa con filtros, iniciar, detener, reiniciar y cambiar tipo de inicio sin abrir el MMC.
🖨️
Impresoras
Impresoras instaladas, colas de impresión, cancelar trabajos, añadir TCP/IP y establecer predeterminada.
📦
Copia de datos
Migración de PC a PC: selección visual, previsualización de tamaño, copia con progreso y verificación final.
📊
Monitor de rendimiento
CPU, RAM, disco y red en tiempo real. Lista de procesos con opción de matar desde la interfaz.
🌐
Red y diagnóstico
Adaptadores, IP, DNS, DHCP. Liberar/renovar IP, vaciar caché DNS y test de ping integrado.
🧹
Limpieza del sistema
30+ categorías de limpieza: temporales, logs, caché de navegadores, prefetch, WinSxS y más.
🔍
Buscador de archivos
Localiza archivos por extensión en cualquier unidad y cópialos a otra ubicación con un clic.
💾
Formateo de unidades
Selecciona la unidad y abre el diálogo de formateo nativo de Windows de forma rápida.
📋
Event Viewer
Visor de eventos del sistema con filtros por nivel, fuente, fecha y ID de evento.
🌍
Interfaz en ES / EU
Disponible en español y euskera. Más idiomas en versiones futuras.
Instalación

En marcha en menos
de un minuto

Sin instaladores, sin dependencias externas, sin conexión a internet.

1
Descarga el ZIP
Descarga Mantendu_v1.0.zip y descomprímelo en cualquier carpeta.
2
Ejecuta como Administrador
Clic derecho sobre INICIAR.bat → "Ejecutar como administrador".
3
Se abre el navegador
La herramienta arranca automáticamente en localhost:8080 con Edge o Chrome.
4
Al terminar
Pulsa cualquier tecla en la ventana del lanzador para cerrar el servidor correctamente.
Requisitos

Funciona en cualquier
Windows moderno

🪟 Windows 10 o Windows 11
⚡ PowerShell 5.1+ (incluido por defecto en Windows)
🔑 Ejecutar como Administrador (necesario para gestión de usuarios, formateo, servicios y Event Log)
🔌 Puerto 8080 libre (solo uso local)
🌐 Edge o Chrome instalado (para abrir la interfaz)
